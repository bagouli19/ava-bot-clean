# ❖ AVA Glossaire de test

GPT-4
GPT-4 est un modèle de langage multimodal développé par OpenAI.  
Il peut traiter du texte et des images, et générer des réponses cohérentes sur une grande variété de sujets.

Streamlit
Streamlit est un framework Python open-source qui permet de construire très rapidement des applications web interactives  
pour le data-science ou les prototypes d’IA.

SentenceTransformer
SentenceTransformer est une bibliothèque Python qui fournit des modèles dérivés de BERT  
pour encoder des phrases en vecteurs (embeddings) et mesurer leur similarité.
